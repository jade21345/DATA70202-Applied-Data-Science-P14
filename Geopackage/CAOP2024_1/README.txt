	1. Dataset Overview

	This data package contains the Official Administrative Charter of Portugal (CAOP) version 2024.1, published by the General Directorate of Territory (Direção-Geral do Território, DGT) on 07-08-2025.

	You can access their website by the link: 
	https://snig.dgterritorio.gov.pt/rndg/srv/por/catalog.search#/search?anysnig=CAOP%20%E2%80%93%20Carta%20Administrativa%20Oficial%20de%20Portugal&fast=index

	CAOP is the official geospatial dataset that defines the administrative boundaries of Portugal.

	This dataset includes:

		Mainland Portugal: 
		Continente_CAOP2024_1.gpkg

		Autonomous Region of Madeira (RAM): 
		ArqMadeira_CAOP2024_1.gpkg

		Autonomous Region of the Azores (RAA): 
		ArqAcores_GCentral_GOriental_CAOP2024_1.gpkg
		ArqAcores_GOcidental_CAOP2024_1.gpkg
		
		All files are provided in GeoPackage (.gpkg) format.

	2. Purpose of the Dataset

		This dataset can be used for:

		Mapping Portuguese administrative divisions

		Electoral district visualisation

		Spatial analysis

		Redistricting simulations

		Socioeconomic data integration

		Construction of adjacency matrices

		GIS-based modelling

	3. Files Included(*)

		1️⃣ Continente_CAOP2024_1.gpkg

		Administrative boundaries for Mainland Portugal.
		Includes layers such as:

		Distrito

		Concelho

		Freguesia

		This file covers the 18 mainland electoral districts.

		2️⃣ ArqMadeira_CAOP2024_1.gpkg

		Administrative boundaries for the Autonomous Region of Madeira (RAM).

		3️⃣ ArqAcores_GCentral_GOriental_CAOP2024_1.gpkg

		Administrative boundaries for the Central and Eastern island groups of the Azores.

		4️⃣ ArqAcores_GOcidental_CAOP2024_1.gpkg

		Administrative boundaries for the Western island group of the Azores.

		Both Azores files must be merged to obtain complete coverage of RAA.
